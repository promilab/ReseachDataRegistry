# Research Data Registry for Process Science

This data overview was created as part of the [NFDIxCS](https://nfdixcs.org/) project. It provides an overview of existing public data available for research in process science. The overview can be accessed here: 

https://promilab.github.io/ReseachDataRegistry/ .

Currently the collection contains data from a variety of domains such as: 
- Software Engineering
- Human-Computer Interaction
- Logistics
- IoT
- Business Administration
- Healthcare
- Gaming
- Robotics
- Blockchain Technology

## Data Sources
- BPI Challenges: https://tfpm.compute.dtu.dk/competitions-awards/bpi-challenge
- Process Discovery Contest Collection: https://www.tf-pm.org/competitions-awards/discovery-contest
- Blockchain Data for Process Mining: https://ingo-weber.github.io/dapp-data/
- IoT Data for Process Mining: https://zenodo.org/communities/iopt/records?q=&l=list&p=1&s=10&sort=newest
- OCEL Event Logs Overview: https://www.ocel-standard.org/event-logs/overview/
- Event Logs (XES, OCEL, CSV): https://www.processmining.org/event-data.html
- Event Logs (XES, CSV): https://zenodo.org/records/16268743
